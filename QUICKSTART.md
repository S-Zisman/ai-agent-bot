# Быстрый старт - AI Agent Bot

## Шаг 1: Создайте бота в Telegram

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Придумайте имя для бота (например: "AI Agent Sergey")
4. Придумайте username (например: "sergey_ai_agent_bot")
5. Скопируйте полученный токен (выглядит как: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Шаг 2: Настройте проект

```bash
# Создайте файл .env
cp .env.example .env

# Откройте файл .env и вставьте токен
nano .env
# или используйте любой другой редактор
```

В файле `.env` замените `your_bot_token_here` на ваш реальный токен:
```
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

## Шаг 3: Запустите бота

### Вариант А: Быстрый запуск (рекомендуется)

```bash
./start.sh
```

### Вариант Б: Ручной запуск

```bash
# Создайте виртуальное окружение
python3 -m venv venv

# Активируйте его
source venv/bin/activate  # для Linux/Mac

# Установите зависимости
pip install -r requirements.txt

# Запустите бота
python bot.py
```

## Шаг 4: Настройте команды бота (опционально)

Чтобы команды отображались в меню Telegram:

1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/setcommands`
3. Выберите вашего бота
4. Вставьте этот список команд:

```
start - Главное меню
about - О Сергее Зисмане
programs - Программы и услуги
contact - Контакты
cases - Кейсы и примеры работ
consultation - Записаться на консультацию
```

## Шаг 5: Протестируйте бота

1. Найдите вашего бота в Telegram по username
2. Нажмите "Start" или отправьте `/start`
3. Проверьте все команды и кнопки

## Готово!

Бот работает локально на вашем компьютере.

## Частые вопросы

### Бот не отвечает?
- Проверьте, что процесс `python bot.py` запущен
- Убедитесь, что токен правильный в файле `.env`
- Посмотрите логи в консоли

### Как остановить бота?
Нажмите `Ctrl + C` в терминале, где запущен бот

См. раздел "Деплой на VPS" в [README.md](README.md)
